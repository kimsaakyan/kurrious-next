# Kurious | Front-end Next-js

## Deployed URL

```
https://kurious-front-end.vercel.app/
```

## Node Version

```
node-18.14.2
```

## Installing dependencies

```
npm i
```

## Running the application

```
npm run dev
```

## Linting

### Check linting issues

```
npm run lint
```

### Resolve linting issues

```
npm run lint:fix
```

## Formatting

```
npm run prettier
```
